Welkom op de startpagina

Klaar om GIT te leren gebruiken?

Opdracht: Klik in het menu op pagina's en navigeer naar het paginanummer dat overeenkomt met jouw klasnummer, maak daar een aanpassing (die door de leerkracht gezien mag worden!) en push deze aanpassing terug naar de repository. Zorg ervoor dat je enkel een aanpassing maakt op de pagina dat met jouw nummer overeenkomt, anders kan je conflicten veroorzaken. Bootstrap is al geïmporteerd, je hoeft dus geen bestanden of links extra toe te voegen.

Github repository: https://github.com/NielsBaptist-TM/DemoLes

Benodigde commando's

Git clone: De repo naar jouw lokale schijf clonen.

Git add . : Wijzigingen toevoegen aan je commit.

Git commit -m "Jouw boodschap": Je commit klaarzetten om te pushen.

Git push: Je commit pushen naar de repo.
